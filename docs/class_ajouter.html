Classe Ajouter

Cette classe représente l'ajout d'un objet par un utilisateur. Elle permet de creer un objet Ajouter et de l'utiliser avec les propriétés idObjet, idUtilisateur et dateAjout.

Exemple d'utilisation : $ajouter = new Ajouter(1, 42, '2024-01-01'); echo $ajouter->getIdObjet(); // Affiche 1
