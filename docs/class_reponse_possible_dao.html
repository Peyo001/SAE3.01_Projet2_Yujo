Description détaillée

Classe ReponsePossibleDao

Cette classe permet d'accéder aux données des réponses possibles dans la base de données. Elle utilise la classe Database pour obtenir une connexion PDO.

Exemple d'utilisation : $reponsePossibleDao = new ReponsePossibleDao(); $reponse = $reponsePossibleDao->find(1);
