Description détaillée

Classe QuestionDAO

Cette classe gère les opérations de base de données pour les objets Question. Elle permet de créer, lire, mettre à jour et supprimer des questions dans la base de données.

Exemple d'utilisation : $pdo = Database\getInstance()->getConnection(); $questionDAO = new QuestionDAO($pdo); $question = $questionDAO->findById(1);
