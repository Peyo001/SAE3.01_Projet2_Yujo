Classe Signalement

Cette classe représente un signalement effectué par un utilisateur. Elle permet de créer un objet Signalement et de l'utiliser avec les propriétés id et raison.

Exemple d'utilisation : $signalement = new Signalement(1, 'Contenu inapproprié'); echo $signalement->getRaison(); // Affiche 'Contenu inapproprié'
