Classe Signaler

Cette classe représente un signalement effectué par un utilisateur sur un post. Elle gère la relation entre un utilisateur, un signalement et un post.

Exemple d'utilisation : $signaler = new Signaler(1, 5, 10, '2024-01-15', 'en_attente'); echo $signaler->getIdUtilisateur(); // Affiche 1
