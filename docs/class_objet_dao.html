Description détaillée

Classe ObjetDao

Cette classe permet d'accéder aux données des objets dans la base de données. Elle utilise la classe Database pour obtenir une connexion PDO.

Exemple d'utilisation : $objetDao = new ObjetDao(); $objet = $objetDao->find(1);
