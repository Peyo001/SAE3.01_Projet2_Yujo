Classe Posseder

Cette classe représente la relation de possession entre une room et un objet. Elle permet de créer un objet Posseder et de l'utiliser avec les propriétés idObjet et idRoom.

Exemple d'utilisation : $posseder = new Posseder(1, 42); echo $posseder->getIdObjet(); // Affiche 1
