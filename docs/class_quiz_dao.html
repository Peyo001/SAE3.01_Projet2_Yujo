Description détaillée

Classe QuizDao

Cette classe permet d'accéder aux données des quiz dans la base de données. Elle utilise la classe Database pour obtenir une connexion PDO.

Exemple d'utilisation : $quizDao = new QuizDao(); $quiz = $quizDao->find(1);
