Description détaillée

ControlleurAccueil gère les actions liées à la page d'accueil comme son affichage.

Hérite de la classe Controller pour bénéficier des fonctionnalités de base. Utilise Twig pour le rendu des vues. Utilise la classe Validation pour la validation des données.

Exemple d'utilisation : $controllerAccueil = new ControllerAccueil($loader, $twig); $controllerAccueil->afficher(); // Affiche la page d'accueil
