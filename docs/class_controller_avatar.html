Description détaillée

ControllerAvatar gère les actions liées aux avatars des utilisateurs comme la personnalisation, l'affichage et la suppression.

Hérite de la classe Controller pour bénéficier des fonctionnalités de base. Utilise Twig pour le rendu des vues. Utilise la classe Validator pour la validation des données.

Exemples d'utilisation : $controllerAvatar = new ControllerAvatar($loader, $twig); $controllerAvatar->showCustomizer(); // Affiche l'interface de personnalisation $controllerAvatar->saveAvatar(); // Sauvegarde l'avatar $controllerAvatar->afficherAvatar(); // Affiche l'avatar $controllerAvatar->supprimerAvatar(); // Supprime l'avatar

showCustomizer ( )

Redirige vers la page de connexion si l'utilisateur n'est pas connecté. Récupère l'avatar existant de l'utilisateur s'il y en a un. Passe les données nécessaires à la vue Twig pour le rendu.
