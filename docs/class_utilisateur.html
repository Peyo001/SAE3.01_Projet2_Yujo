Description détaillée

Classe Utilisateur

Cette classe représente un utilisateur de l'application. Elle permet de créer un objet Utilisateur et de l'utiliser avec les propriétés idUtilisateur, nom, prenom, dateNaissance, genre, pseudo, email, motDePasse, typeCompte, estPremium, dateInscription, yuPoints et personnalisation.

Exemple d'utilisation : $utilisateur = new Utilisateur('Doe', 'John', '1990-01-01', 'Homme', 'johndoe', 'john..nosp@m.doe@.nosp@m.examp.nosp@m.le.c.nosp@m.om', 'password123', 'Standard', false, '2024-01-01', 100); echo $utilisateur->getFullName(); // Affiche "John Doe"
