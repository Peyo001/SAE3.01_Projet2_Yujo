Description détaillée

Classe Question

Cette classe représente une question utilisée dans un quiz. Elle permet de créer un objet Question et de l'utiliser avec les propriétés idQuestion, libelle.

Exemple d'utilisation : $question = new Question(1, 'Quelle est la capitale de la France?'); echo $question->getLibelle(); // Affiche 'Quelle est la capitale de la France?'
