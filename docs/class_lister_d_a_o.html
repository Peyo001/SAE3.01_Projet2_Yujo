Description détaillée

Classe ListerDAO

Cette classe gère les opérations de base de données pour la table LISTER. Elle permet de créer, lire, mettre à jour et supprimer des enregistrements d'association.

Exemple d'utilisation : $listerDAO = new ListerDAO(); $listerList = $listerDAO->findByQuestion(42);
