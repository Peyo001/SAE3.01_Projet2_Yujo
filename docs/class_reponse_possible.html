Classe ReponsePossible

Cette classe représente une réponse possible à une question dans un système de quiz. Elle permet de créer un objet ReponsePossible et de l'utiliser avec les propriétés idReponsePossible, libelle et estCorrecte.

Exemple d'utilisation : $reponse = new ReponsePossible(1, '42', true); echo $reponse->getLibelle(); // Affiche '42'
