Description détaillée

Classe de gestion de la connexion à la base de données en utilisant le pattern Singleton.

Cette classe assure qu'une seule instance de connexion à la base de données est créée et fournit un accès global à cette instance.

Exemple d'utilisation : $db = Database\getInstance(); $conn = $db->getConnection();
