Description détaillée

Classe SignalerDao

Cette classe gère les opérations CRUD pour les signalements (table SIGNALER). Elle utilise la classe Database pour obtenir une connexion PDO.

Exemple d'utilisation : $signalerDao = new SignalerDao(); $signalers = $signalerDao->findAll();
