Classe Lister

Cette classe représente la relation d'association entre une question et une réponse Elle permet de créer un objet Lister et de l'utiliser avec les propriétés idReponsePossible et idQuestion.

Exemple d'utilisation : $lister = new Lister(1, 42); echo $lister->getIdReponsePossible(); // Affiche 1
